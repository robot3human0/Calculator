Калькулятор

Вкладка "Calculator"

Бинарные операторы:

  • Плюс: +
  • Минус: -
  • Умножить: *
  • Делить: /
  • Возвести в стпень: ^
  • mod: остаток от деления числа

Унарные функции:

  • sin: синус угла
  • cos: косинус угла
  • tan: танегенс угла
  • asin: арксинус
  • acos: арккосинус
  • atan: арктангенс
  • √: корень
  • ln: натруальный логарифм
  • log: десятичный логарифм
    • унарный минус
    • унарный плюс

Прочее:

  • X: кнопка для добавления X в функцию в поле ввода
  • X=: поле для ввода значения X
  • =: кнопка для итогово расчета вырадения и вывода его на экран
  • AC: очистить поле
  • ⌫: кнорпка стереть по симврльно
  • .: кнопка десятичной точки для вещественных чисел
  • (: кнопка открывающая скобка
  • ): кнопка закорывающая скобка
  • В так же кнопки от 0 до 9

Вкладка "Function Graph"

Построение графиков:

Draw Graph: построение графика с заданными значениями областей
  • x min минимальное значение области определения
  • x max максимальное значение области определения
  • y min минимальное значение области определения
  • y max максимальное значение области определения
  • Для прстроения графика нужно ввести выражегие во вкадке "Calculator", затем во вкладке "Function Graph" ввести значения для оси X и Y затем нажать на кнопку "Draw Graph"
